# ThinkGraph MCP Friction Log

Record friction points when using ThinkGraph via MCP (Claude Code, Claude Desktop, etc.).

## How to log

Add entries with date, what you were trying to do, and what was clumsy/missing/slow.

```
### YYYY-MM-DD — [what you were doing]
**Friction:** [what went wrong or felt clumsy]
**Severity:** low | medium | high
**Implication:** [what tool/change would fix this]
```
