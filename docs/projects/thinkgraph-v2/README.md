## TL;DR for someone landing here cold (re-audited 2026-04-10)

- **Shipped end-to-end:** Phase 0 (unified node model + cleanup), Phase 1B (graph validation incl. `stuck-worker`, cycles, disconnects, unknown steps), Phase 2B (Vectorize semantic search infra), Phase 3 (fan-in / synthesis), Assistant Step 1 (per-node chat + dispatch conversation summaries), Step 3 (canvas presence — pulsing pill + agent activity feed + `data`/`ephemeral` split).
- **Major new capability (post-audit, not in original briefs):** **Block editor + Pi collaboration.** Notion-style `NodeBlockEditor.vue` in the node slideover with Tiptap extensions (`action-button`, `comment-anchor`, `file-diff`). Pi writes into the editor via Yjs (`yjs-page-client.ts`, `page-tools.ts`). Comment threads (`CommentSlideout.vue`, `useNodeComments.ts`, `comment-tools.ts`) with text-selection composer, "Ask Pi" reply flow (`comment-reply.post.ts`), and comment-to-child-node promotion. Analyst emits passage-level findings as comments. `FileDiffBlock.vue` for inline file diff review. `ActionButtonBlock.vue` for Pi-inserted interactive buttons. `create_node` MCP tool replaced with comment-based suggestions.
- **Partial — infra shipped, integration missing:** Phase 1A (MDC renders, no custom components — dormant services that emitted non-MDC are now deleted), Phase 1C (wiki-link parser/validator exists, not wired to save flow, no clickable rendering), Phase 2A (collections + cron + runner exist, D1 migration shipped `0016`, but no scheduled trigger, no canvas inbox UI, legacy `sync-changelogs` not removed), Phase 1 of v2 brief (structured verdicts + skippable steps shipped, stage-scoped MCP tools missing), Phase 2 of v2 brief (markdown built but worker git-commit step is a TODO), Step 2 (Phase 2B unblocks it but no dialectic endpoint).
- **Not started:** Phase 2C (pipeline formalization), Phase 4 meeting/transcript flow, Phase 5 Nuxt Content, Step 4 (background reflection), Pi worker durability outbox.
- **Cleanup status:** Phase 0 fully complete. 15 dormant dispatch services removed (only `pi-agent` remains). Agent log streaming bugs partially fixed (tool_result truncation fixed, text duplication and thinking fragmentation may persist). Graph validation expanded. **Still open:** legacy DB tables (`thinkgraph_decisions`/`thinkgraph_workitems`) physically in D1, 2 orphaned migration files, `sync-changelogs` parallel to Phase 2A runner, slideover doesn't reactively refresh after dispatch, `stuck-worker` validator has no scheduled runner. See `brief.md` status section for the full list.
