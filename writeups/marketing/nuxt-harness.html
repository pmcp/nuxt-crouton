The idea

The CLI is the runtime. The Harness is the app.

Claude Code and Codex are powerful, empty engines. The Harness is the curated application that runs on them — so a new project starts from a battle-tested path, not a blank prompt.

Executes agents, skills, MCP, hooks, worktrees. Brilliant — and completely unopinionated about your stack.

Building blocks

An app, with the parts you'd expect

Familiar software anatomy — just authored for a probabilistic runtime.

ƒ

Skills = functions

Repeatable workflows the agent calls — the curated know-how, written once.

⚙

Agents = services

Orchestrator → decomposer → worker. They spawn, isolate, and ship PRs.

⚡

Workflows = event handlers

A label, a comment, a merge — each fires the right run, hands-off.

🔌

MCP = drivers

Typed access to your DB, docs, and tools — same servers across both CLIs.

📜

AGENTS.md = constitution

One instructions file both tools read. The rules of the road.

✓

Gates = tests

Schema & UI sign-offs, the packages guard, evals over run logs.

Honest about it

Opinionated by design — and every opinion is optional

Crouton ships a full-stack opinion so you start fast. Don't share the taste? Peel it off. The core is just schema → a typed backend — bring any front end you like.

Core · always generated

The typed backend

Drizzle schema · migrations · DB wiring · logical CRUD API · typed composables. Stack-neutral — useful even if you never touch our UI.

Why it holds together

Built like software, because it is

◇ One source, two tools

Author once; emit a Claude Code plugin and a Codex package. The CLIs differ — your conventions don't.

⤿ Zero drift

A regenerate-and-diff CI gate makes divergence impossible. No more hand-copied configs going stale.

▤ Layered & ownable

A base Harness layer + your project layer. Local overrides survive every sync; fixes flow back upstream.

◎ Tested by evals

A probabilistic runtime needs gates + observation. Weekly run-log analysis is the regression suite.

⛁ OSS & self-hostable

Nuxt-native, no mandatory SaaS. Your stack, your Cloudflare, your data.

⚑ Human where it counts

It pings you only to decide — sign-offs and unblocks. Status is silent. Your attention is the scarce resource.
